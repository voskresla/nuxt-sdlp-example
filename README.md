Для локального запуска dev:

```bash
git clone https://github.com/voskresla/nuxt-sdlp-example.git
cd nuxt-sdlp-example
npx yarn
npx yarn run dev
```

Для локального запуска сгенереной статики:

```bash
git clone https://github.com/voskresla/nuxt-sdlp-example.git
cd nuxt-sdlp-example
npx yarn
npx yarn run generate
npx http-serve dist
```
